# Localhost Form Demo

This project is a simple web form that runs on plain HTTP at:

`http://localhost:8000`

## What it includes

- Letters-only input for full name
- Numbers-only input for age
- Email validation
- Numbers-only phone input with exact length
- Date picker
- Radio buttons
- Select dropdown
- Required checkbox

## How to run

1. Open a terminal in this folder.
2. Run:

```bash
python server.py
```

3. Open your browser and go to:

`http://localhost:8000`
